## Simple Blog App

1.  **Clone/Download the code from this repo**

    ```sh
    git clone git@github.com:Lenislaw/gatsby-1st-app-blog.git
    ```

1.  **Install dependencies**

    ```sh
    cd gatsby-1st-app-blog
    npm install
    ```

    or

    ```sh
    cd gatsby-1st-app-blog
    yarn install
    ```
